artihata
================================
artihata is the composition tool for Emoji.

http://artihata.herokuapp.com/

### Setup

#### server side
```sh
brew install leiningen # install clojure and clojure automating tool.
lein ring server
```

#### client side
```
cd js-project
bower install
npm install
```
